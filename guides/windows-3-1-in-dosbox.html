Requirements:

  • DOSBox-X â€“ This is not a hard requirement, as most of this can be done in stock DOSBox. DOSBox-X makes it a lot easier though.
  • A bootable Windows 3.1 install iso
  • S3 Trio64 video drivers for Windows 3.1
  • Sound Blaster 16 drivers
  • An appropriately-sized empty HDD disk image. You can create one via DOSBox-X or use one of the provided images.

Prep:

  1. Create a new folder to hold everything. We will refer to this folder as root throughout this guide.
  2. Download the files from here: https://archive.org/details/windows-3.11-in-dosbox
  3. Place the following files in the root folder:
    • Windows 3.11.iso
    • S3_Trio64V+_Binary-775-785__Win_3.1_English_v._1.70.04_-_PROD.zip
    • SB16W3X.ZIP
  4. If you wish to use one of the provided HDD images, unzip the appropriate hdd-*.zip file, and place the .img file in the root folder. Rename the unzipped .img file to windows-311.img.
  5. Unzip S3_Trio64V+_Binary-775-785__Win_3.1_English_v._1.70.04_-_PROD.zip into root/S3Trio64. You should end up with root/S3Trio64/OEMSETUP.INF, etc.
  6. Unzip SB16W3X.ZIP into root/SB16. You should end up with root/SB16/INSTALL.EXE, etc.

Hard Disk Image creation:

This step is not required if you use one of the provided HDD .img files.

  1. Start up DOSBox-X.
  2. Click on DOS, then Create blank disk image...
  3. Choose a hard disk image size and hit OK.
    • Windows 3.11 supports a maximum of 2GB.
    • Choose an appropriate size for the installation size of the software you intend to install.
  4. Save the image as root/windows-311.img.
  5. Close DOSBox-X.

Installation:

  1. Start up DOSBox-X.
  2. Click on Drive, then C, then Mount a disk or CD image file. Select root/windows-311.img.
  3. Click on Drive, then D, then Mount a disk or CD image file. Select root/Windows 3.11.iso.
  4. Click on Drive, then E, then Mount folder as hard drive. Select the root/S3Trio64 folder.
  5. Click on Drive, then F, then Mount folder as hard drive. Select the root/SB16 folder.
  6. Click on CPU, then Emulate CPU speed, then Pentium 100MHz (~60000 cycles).
  7. At the Z:\> command prompt, enter these commands:
    • D:
    • setup.exe
  8. At the Welcome to Windows for Workgroups 3.11 Setup screen, hit Enter.
  9. Hit C to run Custom Setup.
  10. Hit Enter to select C:\WINDOWS as the install folder.
  11. At the hardware identification screen, go up to Display: VGA and hit Enter.
  12. Scroll down to the very bottom of the list to Other (Requires disk provided by a hardware manufacturer) and hit Enter.
  13. Change the path location from the default A:\ to E:\ and hit Enter.
  14. Select an appropriate screen resolution and color depth and hit Enter.
    • Note that you WILL be able to easily change this later. This selects the initial resolution/color setting when Windows first starts up.
    • For Windows 3.11-era games, 800x600 256 C SF or 800x600 64K C SF are good choices. Maxing out the resolution is usually not a good idea as games were often very specific about what resolution/color mode they needed to run.
  15. Confirm that the Display now shows S3 Trio64V 1.70.04 along with your selected resolution, and hit Enter.
  16. At the Product Number screen, enter whatever name you wish. Leave the Product Number field blank, and click on Continue. Hit Continue again on the confirmation window.
    • You may need to hit Ctrl-F10 for DOSBox-X to capture your mouse. Hit it again to release it afterwards.
  17. At the Setup will perform each optional taskâ€¦ screen, UNCHECK Set Up Printers and Set Up Applications Already on Hard Disk(s). Leave Set Up Only Windows Components You Select CHECKED and hit Continue.
  18. At the These are optional componentsâ€¦ screen, click on Continue.
  19. At the Network Setup screen, click on Continue.
  20. At the To set up Windows correction, Setup needs to modifyâ€¦ screen, ensure that make all modifications for you is selected, and click on Continue. Click OK on the confirmation popup afterwards.
  21. At the Setup can now run a short tutorialâ€¦ screen, click on Skip Tutorial.
  22. At the Exit Windows Setup screen, click on Return to MS-DOS.
  23. You should be back at the command prompt. Enter the following commands:
    • F:
    • install.exe
  24. At the Welcome to the Audio Software installation program screen, hit Enter.
  25. Select Full Installation and hit Enter.
  26. At the The following path will be usedâ€¦ screen, go down to Microsoft Windows 3.1 path : None and hit Enter.
  27. Ensure that C:\WINDOWS is in the box, and hit Enter.
  28. You should be back at the The following path will be usedâ€¦ screen. Ensure that it now says "Microsoft Windows 3.1 path : C:\WINDOWS and hit Enter.
  29. At the The following settings will be usedâ€¦, go down to Interrupt setting : 5 and hit Enter.
  30. Select 7 and hit Enter.
    • The default DOSBox-X sound settings use an IRQ of 7. You can confirm this by clicking on Sound, then Show Sound Blaster configuration. If it is using a different IRQ value, select that one instead of 7.
  31. You should be back at the The following settings will be usedâ€¦ screen. Ensure that it now says Interrupt setting : 7 (or the appropriate value) and hit Enter.
  32. At the red Backup/Skip/Proceed window, go down to Proceed and hit Enter.
  33. At the Software installation completed screen, hit Enter.
  34. You should be back at the command prompt. Enter the following commands:
    • C:
    • cd windows
    • win
  35. Windows will start up and perform some finalization. Click OK on the End of Setup window, and close the Audio Software window.
  36. In Program Manager, click on File, then Exit Windows. If the mouse is not working yet, hit Alt, then F, then X. Click on OK, or hit Enter.
  37. You should be back at the command prompt. Close DOSBox-X.

At this point, you have a clean Windows 3.11 hard disk image, ready to use in DOSBox-X or stock DOSBox. I suggest making a backup copy of this so you can revert to a clean copy at any point.

Usage:

  1. Start up DOSBox-X.
  2. Click on Drive, then C, then Mount a disk or CD image file. Select root/windows-311.img.
  3. Click on CPU, then Emulate CPU speed, then Pentium 100MHz (~60000 cycles).
  4. Enter the following commands at the command prompt:
    • C:
    • cd windows
    • win

Notes:

  • Any changes made to the C drive (ie. installing software) will persist on that specific hard disk image file.
  • If you wish to mount a CD image to install software, click on Drive, then D, then Mount a disk or CD image file. Select the CD image file and install it normally.
